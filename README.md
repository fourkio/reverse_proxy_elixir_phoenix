# ReverseProxy
A reverse proxy built in Elixir and Phoenix.

## Get Started
* Change your production API url in: `web/views/error_view.ex` 
* Load your local server: iex -S mix phoenix.server
* Hit a working API route: localhost:4000/api/v1/users
